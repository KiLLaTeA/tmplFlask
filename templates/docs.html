Документация API

Эта документация предоставляет информацию о том, как использовать эндпоинты API для работы с данными логов.

Получить все логи

Эндпоинт: GET /api/logs

Описание: Возвращает все записи логов.

Ответ: JSON-массив всех записей логов.


[
    {
        "Age": 25.0,
        "Name": "Иван Иванов",
        "Message": "Пример лог-сообщения"
    },
    ...
]

Получить лог по возрасту

Эндпоинт: GET /api/logs/<float:age_current>

Описание: Возвращает запись лога по заданному возрасту.

Параметры:

  • age_current (float) - Возраст, по которому осуществляется поиск в логах.

Ответ:

Если запись найдена:


{
    "Age": 25.0,
    "Name": "Иван Иванов",
    "Message": "Пример лог-сообщения"
}

Если запись не найдена:


{
    "error": "Записи с таким возрастом не найдены!"
}
